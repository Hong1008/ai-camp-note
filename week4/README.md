# Week 4: 데이터 분석 심화 (NumPy & Pandas)

4주차 수업에서는 데이터의 수치적 특성을 이해하고, 복잡한 조건의 데이터를 자유자재로 추출하고 변형하여 실무 수준의 인사이트를 도출하는 기술을 학습합니다.

---

## 📋 주요 실습 및 기술적 현황

### 🟣 Theme 1: NumPy를 이용한 수치 데이터 및 이미지 처리 (2026-04-17)
행렬 연산 라이브러리인 NumPy의 기본 구조를 익히고, 이미지를 수치 데이터로 다루는 실습을 진행했습니다.
- **배열 구조 이해**: `ndim`(차원), `shape`(모양) 속성을 활용한 데이터 구조 파악 및 `reshape`를 이용한 차원 변형.
- **이미지 데이터 핸들링**: 이미지를 RGB 3차원 배열로 변환하여 특정 색상 채널(Green 등)을 제거하거나 슬라이싱하는 기법 학습.
- **픽셀 연산**: 이미지 밝기 조절 및 `np.clip`을 활용한 데이터 오버플로우(Overflow) 방지 처리.

### 🟡 Theme 2: Pandas를 활용한 데이터 탐색 및 필터링 (2026-04-17)
비즈니스 데이터 분석의 필수 도구인 Pandas의 핵심 기능들을 실습했습니다.
- **기술 통계량 확인**: `describe()`를 통해 데이터의 분포 파악 및 `mean`, `median`, `std`의 통계적 의미 정립.
- **축(Axis)의 이해**: `axis=0`(행 방향), `axis=1`(열 방향) 연산의 차이점과 데이터 '압축(Collapsing)' 개념.
- **데이터 필터링**: 불리언 인덱싱, `query()` 문법, `isin()` 등을 활용한 다중 조건 데이터 추출.

### 🟢 Theme 3: 데이터 전처리 및 고급 병합 전략 (2026-04-20)
머신러닝 모델링 전 단계인 데이터 가공과 여러 테이블을 하나로 합치는 실무 기법을 학습했습니다.
- **범주형 데이터 처리**: 가변수(Dummy Variable) 생성 및 **다중공선성(Multicollinearity)** 문제를 예방하기 위한 `drop_first=True` 전략의 이해.
- **데이터 이산화(Discretization)**: 연속형 데이터를 구간별로 나누는 `pd.cut`과 `pd.qcut` 활용법.
- **데이터 병합(Merge & Concat)**: 단순 이어붙이기인 `concat`과 관계형 조인인 `merge`의 차이점 및 다중 테이블 병합(`on`, `how` 옵션) 실습.
- **결측치 고도화 처리**: `fillna`, `ffill`, `interpolate` 등 상황별 결측치 보간 전략 수립 및 `eval()`을 이용한 효율적인 파상변수 생성.

### 🔵 Theme 4: 데이터 요약 및 다차원 집계 분석 (2026-04-21)
데이터를 그룹화하고 다양한 관점에서 통계량을 산출하며, 시선을 사로잡는 시각화 기법을 학습했습니다.
- **Grouping & Aggregation**: `groupby` 연산의 'Split-Apply-Combine' 메커니즘과 `unstack()`을 활용한 구조 변경.
- **Pivot Table & Crosstab**: 엑셀의 피벗 기능을 넘어서는 다차원 데이터 재구조화 및 리포트 생성 실습.
- **Advanced Visualization**: Seaborn을 활용한 통계적 시각화(Heatmap, Boxplot, Regplot, Jointplot 등) 및 인터랙티브 도구(Plotly) 이해.
- **Performance Optimization**: `apply(lambda)` 지양 및 `np.where`, `np.select`를 활용한 벡터화 연산 최적화 전략.
- **Configuration Strategy**: 하드코딩을 배제한 전역 상수(Date Format 등) 관리 및 '선-계획 후-실행'의 분석가 마인드셋 수립.

### 🔴 Theme 5: 단변량 분석 및 수치 기반 데이터 정의 (2026-04-22)
오늘 실습에서는 한 개의 변수를 집중적으로 탐구하는 단변량 분석(Univariate Analysis) 기법과 이를 객관적 수치로 증명하는 과정을 학습했습니다.
- **기초 통계 및 시각화**: `describe()`, 히스토그램(KDE), 박스플롯을 활용한 분포 파악 및 이상치 식별.
- **객관적 데이터 정의**: 왜도(Skewness)와 첨도(Kurtosis)를 이용해 시각적 관찰을 통계적 사실로 객관화하는 프로세스 정립.
- **비즈니스 인사이트**: Carseats 데이터를 활용하여 소득, 가격, 광고비, 인구 통계 변수가 판매량에 미치는 영향을 단변량 관점에서 가설 수립.
- **데이터 문해력(Data Literacy)**: 개별 고객 데이터와 지역 통계 데이터(Aggregate Data)의 차이를 명확히 구분하여 해석하는 논리적 엄밀성 확보.

### 🟣 Theme 7: 범주형 분석 및 종합 EDA 프레임워크 (2026-04-24)
오늘 실습에서는 범주형 변수 간의 관계를 분석하는 기법을 배우고, 4주간 학습한 모든 기술을 결합하여 실전 데이터 분석 리포트를 작성하는 과정을 진행했습니다.
- **범주형 vs 범주형**: 교차표(`pd.crosstab`)와 모자이크 플롯(`mosaic`)을 통한 시각화, 카이제곱 검정(`chi2_contingency`)을 이용한 독립성 검정 실습.
- **수치형 vs 범주형 (KDE Plot)**: `multiple='fill'` 옵션을 활용하여 특정 수치 구간에서의 타겟 변수 비중(확률) 변화를 시각적으로 분석.
- **종합 데이터 탐색 (EDA)**: 서로 다른 통계 검정(T-test, Chi-square) 결과를 P-value 기준으로 통합 정렬하여 퇴사에 영향을 미치는 핵심 변수(Key Drivers) 순위 산정.
- **분석가적 사고**: 통계적 수치(P-value)뿐만 아니라 비즈니스 맥락에서의 실질적 의미를 결합하여 설득력 있는 리포트 구성.
