# Week 1: Python 기초 및 실습

Week 1에서는 파이썬의 기초 문법을 익히고, 이를 활용한 간단한 알고리즘 문제 풀이 및 데이터 시각화 실습을 진행했습니다.

## 📋 학습 내용 요약

### 1. 파이썬 기초 문법
- **환경 확인**: 파이썬 버전 확인 및 환경 설정
- **자료형 및 변수**: 숫자형, 문자열, 리스트 등의 자료형 활용
- **연산자**: 산술 연산자 및 복합 대입 연산자 활용

### 2. 알고리즘 및 로직 실습
실생활 예제와 게임 로직을 파이썬 코드로 구현하는 실습을 진행했습니다.

- **배스킨라빈스 31 게임**: `while` 루프와 조건문을 활용한 게임 로직 구현
- **주사위 상금 계산**: 조건문을 활용하여 주사위 눈금에 따른 상금 산출 로직 구현
- **문자열 처리**: 
  - 알파벳 빈도수 분석
  - 다이얼 전화기 번호 변환 로직
- **다항식 정리**: 문자열로 된 다항식을 분석하여 동류항을 합치고 간소화하는 알고리즘 구현

### 3. 데이터 시각화 및 NLP 기초
텍스트 데이터를 분석하고 시각화하는 기초 실습을 진행했습니다.

- **KoNLPy (Okt)**: 한국어 형태소 분석기를 이용한 단어 토큰화 및 추출
- **WordCloud**: 추출된 단어들의 빈도수를 기반으로 워드클라우드 이미지 생성 및 시각화
